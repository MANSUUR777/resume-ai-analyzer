# AI Resume Analyzer

A full-stack web application that deterministically scores uploaded PDF resumes against a set of SaaS required skills. Features modern UI/UX with smooth transitions using React, Tailwind CSS, Framer Motion, and a Python Flask backend.

## Structure
- `/frontend` - React, Vite, Framer Motion, Tailwind CSS
- `/backend` - Python, Flask, pdfplumber

## Setup Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `python app.py`

## Setup Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
